General Knowledge Quiz

# Description

This is a simple Python quiz program that asks the user four general knowledge questions. The program checks the answers and calculates the final score.

# Features

* Asks 4 questions
* Checks user answers using if-else statements
* Gives 1 point for each correct answer
* Displays the final score

# How to Run

1. Open the Python file.
2. Run the program.
3. Answer the questions.
4. View your final score at the end.
